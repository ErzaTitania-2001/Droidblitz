# Droidblitz
An amphibot which can traverse both in land as well as water

Tech stack : 
 - Languages: Arduino

Components used : 

- ARDUINO UNO R3
- L293D MOTOR DRIVER
- L298N MOTOR DRIVER
- ROBODO TOWER MG995 GEAR:
- HC05 BLUETOOTH MODULE
- JHONSON MOTORS 12V 300RPM
- PROPELLER 90 ANGULAR
